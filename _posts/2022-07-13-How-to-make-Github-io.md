    ## 🧑‍🔧 담당한 기능
    
    - 처음부터 끝까지 모두 혼자서 담당함
    - 시간대별로 특정한 정류장의 승차 인원을 예측하는 ML 모델 개발
    - ML 모델을 Pickle 파일로 dump하여 서버에 저장 후 웹에서 불러오는 기능
    - 메인 및 상세페이지 템플릿 선택 및 수정
    - 메인 화면에서 사용자가 특정한 정류장을 선택하면 해당 정류장의 정보를 출력해주는 상세페이지 출력
    - 상세 페이지의 네비게이션 바 및 서치 박스
    
    ## 💡 ISSUE
    
    1. **효과적인 파생변수를 찾지 못하여 모델의 성능이 좀처럼 오르지 않음**
        - **주요 변수의 요약통계량**을 파생변수로 추가하여 모델의 성능을 획기적으로 향상시킴.
        - 주요 변수는 **요일, 정류장명, 노션명**으로 지정하여 해당 변수의 조합별 **평균, 합계**를 파생변수로 추가함.
        
    2. **각각 모델의 사이즈가 1GB를 넘어가며, 하이퍼 파라미터 튜닝시간이 10시간을 넘김.**
        - **LightGBM**으로 학습하여 해결
        - 용량과 학습시간 모두 단축되었고 성능(RMSE, R2)도 향상됨.
        
    3. **가상환경에서 Pycaret을 설치하려고 하면 Numpy와 버전 충돌이 발생한다는 에러와 함께 install 실패**
        - 가상환경을 기존 venv 모듈에서 **conda**로 변경하여 해결
        - 따로 numpy와 pandas의 버전을 변경하지는 않고 pandas-profiling 3.1.0버전을 설치
